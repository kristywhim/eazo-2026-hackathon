角色视图
🎙 主持人 Host — 你的核心职责
Day 1：开幕热场 → 介绍 Founder → 串联赛道介绍 → 各餐时分享节点广播 → 提交截止预警广播
Day 2：热场 → 主持圆桌 Panel（严格30min内结束）→ Demo规则宣讲 → 每队 Demo 计时+呼叫 → 超时打断 → 休息节点广播 → 投票预警 → 颁奖全程串场
关键原则：你是节奏的守门人。任何环节超时优先打断，礼貌但坚定。
🙋 Volunteer — 你的核心职责
签到组：核对名单 → 发胸牌 + 赛道贴纸 → 引导入座 → 处理临时到场
场务组：桌椅布置 / 标识上墙 / 餐饮摆台 / 通宵零食补给
社媒组：各分享节点现场示范发帖 → 截图优质分享发至运营群
舞台组：话筒传递 / 举时间提示牌（1min / STOP）/ 管理 Demo 队列 / 线上等候室
关键原则：主动！不要等参赛者来找你。每个分享节点主动走到参赛者旁边帮他们发帖。
🧑‍🏫 Mentor 导师 — 你的核心职责（按赛道负责）
负责范围：你的赛道分区 — 主动巡场，不要等选手来问你
Build阶段：每30–45分钟在赛道区走一圈 → 了解各队进展 → 给产品方向建议 → 鼓励尽早提交
你不做的事：不参与评分，不替选手做产品决策，不给技术实现建议（仅平台操作可帮）
关键信号：如果某队卡住超过1小时没有进展，及时告知运营负责人
Day 2：可选参与，观看自己赛道的 Demo；无需出席颁奖
⚖️ Judge / Guest 评审嘉宾 — 你只需要出席 Day 2
到场时间：Day 2 09:30（30分钟前到场，领取评审礼包）
10:00–10:30：上台参与圆桌 Panel 论坛（主持人主导节奏，你只需准备1–2句开场观点）
10:30 起：入座评审席（前排中央），全程观看 Demo，每队打分
评分维度：创新性 / 用户价值 / 完成度 / 商业潜力 / 现场表现
颁奖环节：上台为获奖项目颁奖 / 见证徽章揭晓
关键原则：打分不等投票截止，随时可提交；如需提前离场请提前告知运营负责人
